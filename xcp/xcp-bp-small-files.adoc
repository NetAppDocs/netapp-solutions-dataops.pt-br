:allow-uri-read: 
:nofooter:

[role="lead"]
Este caso de uso é baseado no maior cliente do setor de turismo da NetApp para migração de dados locais para a nuvem.  Como a COVID-19 reduziu a demanda no setor de viagens, os clientes querem economizar despesas de capital em armazenamento de ponta em seu ambiente local para a aplicação de preços sob demanda.  Este cliente tem um SLA rígido para migrar milhões de arquivos pequenos para a nuvem.

A figura a seguir descreve a migração de dados do local para o Azure NetApp Files para arquivos pequenos.
